
# Lootbox Demo Starter

## Features
- Fake balance
- Lootbox opening
- Admin logs
- Backend API
- Frontend React app

## Start Backend

```bash
cd backend
npm install express cors
node server.js
```

## Start Frontend

Use Vite or Next.js and paste App.js.
